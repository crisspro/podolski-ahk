Ayuda de Podolski-AHK

Script para usar el plugin vSTI Podolski con lectores de pantalla.

probado con:

  • Podolski 1.2
  • Reaper 6.11
  • Windows10 64bits

Licencia GPL-3.0

Código fuente y últimas versiones:

https://github.com/crisspro/podolski-ahk

Cómo usarlo

  1. Ejecuta Podolski-AHK.
  2. abre Podolski desde tu secuenciador.
  3. Usa los atajos de teclado del script.
  4. Puedes cerrar el script desde la bandeja del sistema

Atajos de teclado

Atajo Descripción
A Cambia al preset anterior
S Cambia al siguiente preset.
p Despliega el menú con los presets.
F1 Abre la Ayuda del script
CTRL+q Sale del script
