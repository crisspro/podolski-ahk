Podolski-AHK help

Script to use Podolski VSTI with screen readers.


    This script maybe be working with any DAW, was tested with:

  • Podolski v1.2.
  • Reaper 6.11.
  • Windows 10 x64.

Licence GPL-3.0

Source code and last releases:

https://github.com/crisspro/podolski-ahk

Usage

  1. Run Podolski-AHK
  2. Open Podolski in your DAW.
  3. Press the hotkeys of script.
  4. Can close the script from system tray

Hotkeys

Hotkey Description
P Open the presets menu.
S Next preset.
A Back preset.
F1 Open help documentation.
CTRL+q close the script.
